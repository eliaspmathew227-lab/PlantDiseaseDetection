# Project Documentation

Use these files for project submission and presentation:

- `PROJECT_REPORT.md`: Full report draft for B.Tech project documentation.
- `PRESENTATION_OUTLINE.md`: Slide-by-slide presentation content.
- `DEMO_SCRIPT.md`: Spoken demo flow and viva questions.
- `TECHNICAL_SETUP.md`: Local and Railway deployment setup notes.

Recommended workflow:

1. Use `PROJECT_REPORT.md` as the base for the formal report.
2. Convert `PRESENTATION_OUTLINE.md` into PowerPoint slides.
3. Practice the demo using `DEMO_SCRIPT.md`.
4. Keep `TECHNICAL_SETUP.md` for deployment troubleshooting.
